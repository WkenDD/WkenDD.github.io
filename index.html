About Me
My introduction

Hai, nama saya Grehista! Saya adalah seorang pelajar yang gemar menggeluti dunia Minecraft. Sebelumnya, saya pernah menciptakan server Minecraft bernama SankuProject, meskipun sekarang sudah tidak aktif lagi. Saya juga pernah bergabung dalam beberapa server lainnya, termasuk BTA (Brutal Theft Auto), meskipun hanya untuk waktu yang singkat. Selain itu, saya memiliki hobi dalam fotografi dan video, serta suka mengedit video. Kemampuan saya juga termasuk membuat website, meskipun masih dalam tingkat dasar. Saya selalu bersemangat untuk terus belajar dan mengembangkan keterampilan saya di bidang-bidang ini!
